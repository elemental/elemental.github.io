##########################
08/04/2011: Elemental 0.68
##########################

`elemental-0.68.tgz <http://libelemental.org/pub/releases/elemental-0.68.tgz>`__

Changes
=======
- Extended the AxpyInterface to allow for both the global-to-local and local-to-global directions
- Extended print functions to allow for arbitrary output streams rather than just assuming std::cout.
- Fixed a potential bug with r-value strings being passed by reference.
