##########################
10/25/2011: Elemental 0.70
##########################

`elemental-0.70.tgz <http://libelemental.org/pub/releases/elemental-0.70.tgz>`__

Changes
=======
- Avoided a bug in PMRRR's global eigenpair sorting routine
- Added advanced::SortEig for sorting the eigenpairs
- Modified eigenvalue vector to store columnwise rather than rowwise
- Added the ability to 'View' buffers after (Dist)Matrix construction
- Added custom exceptions to aid an upcoming 'advanced::Determinant' routine
- Greatly improved the documentation (though it is far from finished)
