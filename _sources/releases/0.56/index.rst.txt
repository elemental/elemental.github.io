##########################
01/07/2011: Elemental 0.56
##########################

`elemental-0.56.tgz <http://libelemental.org/pub/releases/elemental-0.56.tgz>`__

Changes
=======
Improvements from version 0.55 include a bug fix for hybrid PMRRR and simplification of the BLAS/LAPACK underscore convention handling. PMRRR was also modified to avoid using MKL for its hybrid build due to apparent issues.
