##########################
01/13/2011: Elemental 0.58
##########################

`elemental-0.58.tgz <http://libelemental.org/pub/releases/elemental-0.58.tgz>`__

Changes
=======
- Significant reduction in memory usage of eigensolvers
- Performance improvement for Symm and Trmm for small numbers of RHSs
- Another bug fix for UT transforms applied to nonsquare matrices
- The beginnings of a C interface
