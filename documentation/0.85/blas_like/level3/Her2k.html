Her2k¶

Hermitian rank-2K update: updates \(C := \alpha A B^H + \bar \alpha B A^H + \beta C\), or \(C := \alpha A^H B + \bar \alpha B^H A + \beta C\), depending upon whether orientation is set to NORMAL or ADJOINT, respectively. Only the triangle of \(C\) specified by the uplo parameter is modified.

Please see SetLocalTrr2kBlocksize<T>() and LocalTrr2kBlocksize<T>() in the Tuning parameters section for information on tuning the distributed Her2k().

Note

For the best performance, A, B, and C should all be in [MC,MR] distributions.
