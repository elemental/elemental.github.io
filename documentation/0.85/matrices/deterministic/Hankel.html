Hankel¶

An \(m \times n\) matrix \(A\) is called a Hankel matrix if there exists a vector \(b\) such that

\[A(i,j) = b(i+j).\]

The following routines create an \(m \times n\) Hankel matrix from the generate vector, \(b\).
