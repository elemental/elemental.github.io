Cauchy¶

An \(m \times n\) matrix \(A\) is called Cauchy if there exist vectors \(x\) and \(y\) such that

\[A(i,j) = \frac{1}{x(i) - y(j)}.\]

The following functions generate a Cauchy matrix using the defining vectors, \(x\) and \(y\).
