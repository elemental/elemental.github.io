Affine Linear Programs¶

\[\begin{split}\min_{x,s} & \{\; c^T x \; | \; A x = b \;\wedge\; G x + s = h\;\wedge\; s \ge 0 \;\}, \\ \max_{y,z} & \{\; - b^T y - h^T z \; | \; A^T y + G^T z + c = 0 \;\wedge\; z \ge 0 \;\}\end{split}\]
